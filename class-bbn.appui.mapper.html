Class mapper

This class builds special tables and defines databases' structure and according forms in them. The built tables all have the same prefix (bbn in this example), and are called:

  • bbn_projects
  • bbn_dbs
  • bbn_tables
  • bbn_columns
  • bbn_keys
  • bbn_forms
